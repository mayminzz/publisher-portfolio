About me

사용자의 입장에서 한번 더 생각하는 웹 퍼블리셔가 되고 싶습니다.

지속적인 학습을 위해 스터디와 온라인 강의에 적극 참여하고, 낯선 기술에도 포기 하지 않고 해결하고자 노력하고 있습니다. 열린 마음으로 피드백을 소중히 여기고 타인의 의견을 존중하며, 경청할 준비가 되어 있습니다.

Skill

HTML5

웹표준을 준수하고, 구조에 따라 적절히시맨틱 태그를 사용하여 마크업 할 수 있습니다.

CSS3/SCSS

구조에 따라 어울리는 레이아웃으로 flex, grid 등을 사용하여 배치하고, 폰트와 색상 스타일을 적용합니다. 미디어쿼리로 반응형 웹을 구현합니다.

JavaScript

객체, 배열 등의 자료구조를 효율적으로 사용, 클래스 및 함수를 활용한 기능구현이 가능합니다. 외부 API 데이터를 활용한 프로젝트 경험이 있습니다.

Jquery

자바스크립트의 라이즈러리인 제이쿼리 문법에 대해 이해하고 코글 메뉴 등의 기능을 구현합니다. 또한 플러그인 사용이 가능합니다.

React

ToDolist 등의 간단한 프로젝트를 진행해보았고, 영화데이터 등 외부 API를 활용한 CRS의 장점을 이해하고 있습니다.

git/slack

작업물을 vsCode로 git에 업로드하고, 공유할 수 있습니다. git과 slack으로 협업합니다.

Firebase

firebase를 통해 홉페이지를 배포하고 수정할 수 있습니다.

Design Tool
Illustrator/ photoshop/ Figma

아이콘 제작, 사진 수정과 편집이 가능하고, 디자인 시안을 만들수 있습니다.
